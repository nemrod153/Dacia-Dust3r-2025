# Dacia Dust3r

Repo pentru codurile echipei Dacia Dust3r în pregătire pentru NWERC 2025 Karlsruhe


### Caiet tehnic:

- Graph Algorithms
  - Breadth-First Search;
  - Depth-First Search;
  
  - Single-Source Shortest Path:
    - Lee;
    - Dijkstra;
    - Bellman-Ford;
    - A*;
    
  - All-pairs Shortest Path:
    - Floyd-Warshall;

  - Connected Components:
    - Union-Find;
    - Kosaraju;
  
  - Minimum Spanning Tree:
    - Kruskal;
    - Prim;


- Concepte teoretice
    - C++ types and modifiers
      - primitives; int8_t; int_fast32_t;
      - modifiers (const, constexpr, static_cast);
      - expresii utile (std::optional<T\>, std::numeric_limits <T\> și parametrii)
